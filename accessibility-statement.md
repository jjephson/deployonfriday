# Accessibility Statement

## Compliance Status

This website is designed to be accessible and usable by all users, including those with disabilities. We are committed to ensuring digital accessibility for people with disabilities and are continually improving the user experience for everyone.

## Standards Compliance

This website strives to conform to the following accessibility standards:

- **WCAG 2.1 AA** (Web Content Accessibility Guidelines 2.1 Level AA)
- **EAA** (European Accessibility Act) requirements
- **EN 301 549** (European standard for digital accessibility)
- **Section 508** (US federal accessibility standards)

## Accessibility Features Implemented

### 1. Semantic HTML Structure
- Proper heading hierarchy (H1, H2, H3)
- Semantic HTML5 elements (`<main>`, `<section>`, `<nav>`, `<header>`, `<footer>`)
- ARIA landmarks and roles for screen reader navigation
- Proper list structures with `role="list"` and `role="listitem"`

### 2. Keyboard Navigation
- Skip link to main content for keyboard users
- All interactive elements are keyboard accessible
- Visible focus indicators with 2px outline
- Logical tab order throughout the site

### 3. Screen Reader Support
- Descriptive alt text for all images
- ARIA labels and descriptions where needed
- Visually hidden headings for screen reader context
- Proper link descriptions with context

### 4. Visual Accessibility
- High contrast color schemes (4.5:1 minimum ratio)
- Support for `prefers-contrast: high` media query
- Dark mode support with `prefers-color-scheme: dark`
- Scalable text and responsive design

### 5. Motion and Animation
- Respects `prefers-reduced-motion` user preference
- Disables animations and transitions when requested
- Smooth scrolling can be disabled for users who prefer it

### 6. Touch and Mobile Accessibility
- Minimum 44px touch targets for all interactive elements
- Responsive design that works on all screen sizes
- Proper viewport meta tag for mobile devices

### 7. Color and Contrast
- WCAG AA compliant color contrast ratios
- Color is not the only means of conveying information
- Support for high contrast mode
- Dark and light theme support

### 8. Typography and Readability
- Clear, readable fonts (Inter font family)
- Appropriate line height (1.6-1.7)
- Proper letter and word spacing
- Scalable text that works with browser zoom

## Testing and Validation

This website has been tested with:
- Screen readers (NVDA, JAWS, VoiceOver)
- Keyboard-only navigation
- Browser zoom up to 200%
- Various assistive technologies
- Automated accessibility testing tools

## Known Issues

Currently, there are no known accessibility issues with this website. If you encounter any accessibility barriers, please contact us using the information provided in the contact section.
